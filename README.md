# webrtc-platform

Для установки iOS требуется XCode 14.2, Cocoapods.
1. Открыть в терминале папку SupportWebRTC
2. pod install

Для запуска сервера
1. Открыть в терминале папку SignalingServer
2. npm install
3. node server.js
